# 基座模板：所有角色共享的蒸馏维度

> 任何角色都需要提取的通用维度。具体角色模板在此基础上增加特有维度。

## 通用蒸馏维度

### 1. 互动风格

- 默认沟通方式（长/短、结构化/自由、文字/语音偏好）
- 语气基调（正式/随意/幽默/严肃/变化模式）
- 响应节奏（秒回/深思熟虑/视场景而定）

### 2. 口头禅与语言特征

- 高频词汇与表达习惯
- 特有的比喻、类比、举例风格
- 语言转换模式（何时切换正式/非正式）

### 3. 价值观与原则

- 反复强调的理念或信条
- 面对利益冲突时的优先级排序
- 对"好"与"不好"的判断标准

### 4. 决策模式

- 做选择的依据（数据/直觉/共识/权威）
- 面对不确定性的默认策略
- 改变主意的触发条件

### 5. 情绪表达

- 高兴/赞赏时的表现方式
- 不满/反对时的表达模式
- 压力下的行为变化

## 证据标注规则（所有维度通用）

| 标签 | 含义 | 使用场景 |
|------|------|---------|
| `verbatim` | 可指回具体消息/文件的摘抄或原话 | 聊天记录、邮件原文、社交媒体帖子 |
| `artifact` | 文档/作品/公开内容中的客观陈述 | 文章、代码、演讲、采访 |
| `impression` | 提供者补充的主观印象 | 「我觉得 TA...」 |

## 伦理基线（所有角色必须遵守）

1. 产出物是**辅助理解与对齐工具**，不构成对真人身份的冒充授权。
2. 涉及可识别的第三方真名或隐私信息，建议改用代号。
3. 产出物中必须包含伦理声明，标明来源与局限。
4. `impression` 级别的内容必须与 `verbatim`/`artifact` 隔离存放。
